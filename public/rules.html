How To Play

Pick 6 is a season-long competition where users compete against their group trying to accumulate as many points as possible.

Points are accumulated by picking a team's spread or moneyline for that given week.

Can make up to 6 selections each week with an optional 7th pick.

Certain picks have different point values.

Point System

  • Push: +0.5 pts
  • Favorited moneyline win: +1 pt (points change to +0.5 pts if ML is -250 or higher)
  • Spread win: +1.5 pts
  • Underdog moneyline win: +2 pts (points change to +2.5 pts if ML is +250 or higher and +4 pts if ML is +400 and higher)
  • - Points will be awarded after each slate of games

Immortal Lock

An optional 7th pick per week that can either gain users +1 pt with a win or -2 pts with a loss

A spread or moneyline of any value will not alter the amount of points that can be gained or lost

Ex: -400 ML earns the same amount of points as a +400 ML and no matter the line, a loss will always subtract 2 points

A user must toggle on the “Set Immortal Lock” button in order to select an immortal lock for that week

The toggle button can be found in the “Pick Selection Page” and the immortal lock will show up as the users 7th pick

Bi-Weekly Selections

When making selections keep in mind that the same pick can not be used in consecutive weeks

Meaning that users can not select the Bengals spread 2 weeks in a row

If a user selects the Bengals spread in week 5, that selection will be unavailable until week 7. However, Bengals Moneyline will be available for selection in week 6 and vice versa

Deadlines

Picks open each week at 12am on Tuesday and close at 7pm on Thursday and can be made on the “Pick Selection Page”

At 7pm on Thursday the “Pick Selection Page” will lock, selections will not be allowed and can not be altered. At that time all user selections for that week will be visible to the rest of the group

Pool Manager

In the Pool Manager, users can create new pools or join existing public and private ones.

Pool Creation Options: When creating a pool, users can set it to Public (open to all) or Private. Private pools require setting a password (Note: Passwords are case sensitive!). Users must also choose the game mode: Classic or Survivor. For Classic mode pools only, there is an option to enable Playoffs.

Classic Mode Playoffs (Optional):

If the playoff option is enabled for a Classic pool, a head-to-head (h2h) bracket competition will begin in Week 14.

The top 10 users in the pool, seeded based on their rankings at the end of Week 13, will qualify for the bracket.

For users who do not qualify for the playoffs, their regular season effectively ends after Week 13, but they can still watch the action unfold.

The playoffs continue through the subsequent weeks, concluding at the end of Week 17.

Requirement: The playoff feature can only be enabled for pools containing 6 or more users.

Other Pool Management: Pool Admins have the capability to delete the pools they manage. All users are provided with the option to leave any pool they have previously joined.

Homepage Order: The Pool Manager also allows users to reorder how their joined pools are displayed on their homepage. This can be done by clicking the up and down arrows located next to each pool name within the manager.

Survivor Mode

Pick 6's Survivor Mode is a high-stakes, elimination-style competition separate from the classic points-based game. The objective is simple: the user must survive longer than everyone else in their pool.

Each week, the user must select exactly one pick that they believe will win outright (straight up, ignoring the point spread).

If the pick the user selects wins, they advance to the next week. If the pick the user selects loses or ties, they are eliminated from the Survivor pool for the season.

A user's pick can only be used ONCE throughout the entire season. For example, if a user picks the Chiefs in Week 1, they cannot pick them again in any subsequent week for the rest of the season in that specific Survivor pool.

The last user remaining who has not been eliminated wins the Survivor pool.

There is a locking feature at the top of the pool for admins to prevent any other users from joining the pool, regardless of if it's public or private. While that is toggled, it will be impossible for other users to join.

This is implemented to ensure the survivor pool cannot be infiltrated after the group decides when to start.

Upon joining or creating a survivor pool, the "Survivor Selection Page" button will appear on the top of the user's homepage. This page is specifically for making their weekly Survivor pick, separate from the classic Pick 6 selection page.

Deadlines for making their Survivor pick each week are the same as the classic mode (locking Thursday at 7 PM).
